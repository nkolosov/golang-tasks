# Вычисление числа Пи
Реализовать многопоточное вычисление числа Пи при помощи [ряда Лейбница](https://ru.wikipedia.org/wiki/%D0%A0%D1%8F%D0%B4_%D0%9B%D0%B5%D0%B9%D0%B1%D0%BD%D0%B8%D1%86%D0%B0)

## Требования
- Через параметры задаем требуемое число воркеров (каждый воркер получает только свои элементы ряда, по умолчанию - 1 воркер)
- Через параметры можем указать желаемую точность (число знаков после запятой при выводе, по умолчанию 7)
- При получении сигнала `SIGINT` / `SIGTERM` программа должна завершиться и отобразить полученный результат
- Можем указать тайм-аут (программа должна завершиться по его истечению, если не указан - ждём получения сигнала `SIGINT`
- При включении расширенного вывода (`--verbose`) также необходимо вывести дополнительные данные (общее число итераций и общее время работы программы)
- Обязательно написать benchmark-тест (`testing.B`)

Пример запуска
```shell
$ /your/app [--workers 100] [--precision 2] [--timeout 5s] [--verbose]
3.14
```
