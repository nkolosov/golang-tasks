# Множество

## Описание задачи

Нужно реализовать пакет и тесты для работы со структурой данных "Множество"

У множества должны поддерживаться следующие операции:
- Добавление элемента в множество
- Удаление элемента из множества
- Проверка наличия элемента в множестве
- Пересечение двух множеств
- Объединение двух множеств
- Вычитание множества

## Запуск проекта
Работающий `main.go` не требуется
