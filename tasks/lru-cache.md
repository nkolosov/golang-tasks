# LRU Cache

## Описание задачи

Нужно реализовать пакет и тесты для работы с кешом в логике LRU

У кеша должны быть реализованы следующие методы:
- `Get` возвращает элемент по его ключу
- `Set` сохраняет значение в кеше по его ключу
- `Clear` очищает кеш

## Запуск проекта
Работающий `main.go` не требуется
