# Занятия

## Занятие 1. Базовые типы данных.
Стандартная библиотека. Пакет builtin. Типы данных.
Структуры. Пустые структуры. Интерфейсы. Пустые интерфейсы.
Встраивание структур. Агрегация и композиция. Интерфейсы. 
Выравнивание структур. Value / Pointer Receiver.
Модель памяти: Стек и Куча. Escape Analysis.

## Занятие 2. Массивы и срезы.
Slice. Отличия массивов и слайсов. Адресная арифметика.
Внутреннее устройство slice.
Команды make / append / copy. 
Структуры данных. Стек. Очередь. Односвязный и двусвязный список.
Строки. Особенности работы со строками в Go. 
Байты. Руны. Кодировка. Unicode. UTF-8.

## Занятие 3. Hashmap
Map. Внутреннее устройство. 
Хэширование. Хэш-функция. Коллизии. Алгоритмы разрешения коллизий. 
Бакеты. Swiss Table. Особенности работы с Map. Эвакуация.
Практическое применение.

## Занятие 4. Go Runtime. Работа ОС.
Процессы и потоки ОС. Планировщик ОС.
Память процессов и потоков. Context Switching.
Взаимодействие процессов. Способы взаимодействия процессов.
Различные модели многозадачности. Различные профили нагрузки. CPU-bound. IO-bound.
Zombie-процессы. Orphan-процессы.

## Занятие 5. Go Runtime
Go Runtime. Составные части. 
Go Scheduler. Garbage Collector. Memory Allocator. Stack Manager.
Go Scheduler. GMP-модель. Горутины. Внутреннее устройство горутин.
Стек и куча. Сборщик мусора. Mark & Sweep. Трехцветный алгоритм. Write Barrier.

## Занятие 6. Синхронизация горутин.
Каналы. Внутреннее устройство каналов. Аксиомы каналов.
Мультиплексирование (select). 
Контексты. Инструменты синхронизации. Пакет sync. Пакет /x/sync.
Мьютексы, семафоры, atomic. 
Паттерны синхронизации worker pool, fan-in, fan-out.

## Занятие 7. Обработка ошибок. Отложенная обработка
Ошибки. Работа с ошибками в Go. Паттерны многопоточной обработки ошибок.
errors.New / fmt.Errorf
errors.Is / errors.As, Работа с errgroup.
Паника. Конструкция defer. Функция recover().
Паттерны Must...()
Логирование, трассировка (OpenTracing), OpenTelemetry, Sentry.
Метрики (Prometheus). SLA / SLI / SLO. 
Паттерны Health Check. Circuit Breaker.
